# NGO Awareness Webpage

A simple and responsive NGO awareness webpage created using HTML and CSS.  
This project was developed as part of an internship task to spread awareness about NGO initiatives, social impact, and ongoing projects.

## Project Objective

The main objective of this project is to create a clean and informative webpage that highlights:

- NGO introduction
- Ongoing social projects
- Social impact and mission
- Visual content and gallery
- Call-to-action for volunteers and supporters

## Features

- Responsive webpage design
- Clean UI with proper section alignment
- NGO introduction section
- Ongoing projects section
- Social impact section
- Image gallery
- Join Us / Volunteer call-to-action
- Simple navigation bar

## Technologies Used

- HTML5
- CSS3
